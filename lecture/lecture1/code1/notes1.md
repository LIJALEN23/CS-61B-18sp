# Essentials
## Hello world 
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("HelloWorld!");
    }
}
```
1. 一个HelloWorld的类中
2. main方法为入口
3. String[] args : 命令行参数
4. System.out.println("");进行输出
